    Aircraft Damage Classification & Captioning

      Tech Stack: Python, TensorFlow, Keras, HuggingFace Transformers (BLIP), Matplotlib, PIL

      A computer vision project that classifies aircraft images as “Damaged” or “Undamaged” and generates descriptive captions using a pretrained transformer-based model. Originally developed as part of IBM’s Deep Learning Capstone, the notebook was fully refactored into modular, production-style Python code with added visualizations and performance enhancements.

      • Image classification using a fine-tuned VGG16 CNN model
      • Image captioning with HuggingFace's BLIP (Bootstrapped Language-Image Pretraining)
      • Refactored IBM’s original Jupyter Notebook into a clean, object-oriented Python script
      • Training performance visualizations (loss/accuracy curves, sample predictions)
      • Modular folder structure for data, models, inference, and utilities

      Created as: Final capstone for IBM Deep Learning with Keras Specialization
